About Me

I'm Swaraj Jagdale, a Software Engineer II at Dell Technologies, building agentic AI systems and distributed backend services for enterprise data protection platforms that safeguard petabyte-scale workloads across Fortune 500 customers.

My current work centers on agentic AI — designing multi-step, stateful LLM workflows using LangGraph and MCP (Model Context Protocol) servers to orchestrate complex reasoning across Dell PowerProtect Data Manager. I bridge AI agents with production microservices, turning research-grade ideas into reliable, observable systems.

I work full-stack: Java and Python for backend services, system design, and AI integrations; Angular and TypeScript for scalable frontends; and Jenkins-driven CI/CD pipelines that ship code reliably. I care deeply about clean architecture, performance, and production-grade quality — backed by strong test automation using Cypress and JUnit.

On the research side, I hold an M.Tech in Computer Science from NITK Surathkal, with published work on graph-based transformers for multi-modal visual reasoning. My broader interests span LLM agents, RAG systems, explainable AI, and structured reasoning — and how to apply them to real engineering problems.

I'm driven by long-term growth as an engineer and researcher, with a focus on building intelligent systems that combine thoughtful design with measurable impact.
